API de serviços do TCC

Segue abaixo os endpoints dos serviços

Dados

GET Todos os dados do sensor

http://umonitor-api.herokuapp.com/dados/todos
Example Request Todos os dados do sensor 
curl --request GET \
  --url http://umonitor-api.herokuapp.com/dados/todos

POST Criar dados de sensor

http://umonitor-api.herokuapp.com/dados/sensor/:idSensor

Headers

Content-Type application/json

Body

raw (application/json) 
 {
    "T":2,
    "E": true
}
Example Request Criar dados de sensor 
curl --request POST \
  --url http://umonitor-api.herokuapp.com/dados/novo \
  --header 'Content-Type: application/json' \
  --data ' {
    "temperaturaAtual":2,
    "dataAtual": "2018-04-24T18:04:22Z",
    "temEnergia": true,
    "sensorId": 2
}'
